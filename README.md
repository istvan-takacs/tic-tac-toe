# Tic Tac Toe

A browser-based Tic Tac Toe game built as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-tic-tac-toe) JavaScript curriculum.

## Features

- Play as X or O against a bot
- Bot logic: wins if possible, blocks if needed, otherwise plays randomly
- Score tracking across rounds (ties award both players a point)
- Starting player alternates each round

## Built With

- HTML, CSS, JavaScript
- Factory functions and the module pattern (IIFE)
- Separation of concerns: `Gameboard`, `GameController`, and `DisplayController`
